                                Complile Successfully in Windows 7 with Microsoft Visual C++ 2015 and Boost Library l.55

yftpserverandroid/YFtpServerWapper.sln----------------Microsoft Visual C++ 2015 + VisualGDB 5.0 Solution File for Android
